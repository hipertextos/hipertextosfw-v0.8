Cookies

jquery.cookie.js

jquery.cookie.js are a jquery plugin created by Klaus Hartl and distributed under the MIT license. With a few lines of javascript code, it allows to manage cookies. jQuery Cookie are included in the scripts.js file.

With this included plugin it's easy to add, delete and set browser's cookies. More info at jQuery Cookie.

Sample

jquery.cookie.js is used by scripts.js to set custom views and keep in cache. You can see in action in the #menu or #viewer components.
