Managing icons

SVG management with jQuery

hipertextos framework includes a SVG icons collection from Flaticon service. Icon size has been regularized inside a 512 pixels square.

All SVG files are loaded from a javascript file: icons.js. Then, it's possible get to control the innards of the SVG with CSS. The inline SVG can combine with color classes included in the styles file.

Use the .icon class to determine the size of the icon
